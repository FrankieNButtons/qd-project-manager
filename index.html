Get started in 3 steps

1

Clone & install

Clone the repo and run npm install. Requires Node.js 18+. No native toolchain needed.

2

Add your teams

Open the Settings tab, paste a Tencent Docs spreadsheet link, and give the team a name. Repeat for each team.

3

Watch it work

The app fetches data automatically, ranks teams, and starts watching for indicator changes. You're done.

Ready to track?
